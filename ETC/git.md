
### git 명령어
```
$ git status
$ git init   
$ git add -A  
$ git commit -m "코멘트"
$ git reset 돌아갈커밋번호 --hard
$ git revert 커밋번호6자리  // history는 남기고 돌아감
$ git branch 브랜치이름 //브랜치생성
$ git checkout 브랜치이름  // 브랜치이동
$ git checkout -b 새브랜치이름 // 브랜치 생성, 체크아웃
$ git merge 머지할브랜치이름
$ git rebase 머지할브랜치이름 // 재배치, 분기깔끔쓰

$ git push origin master
$ git fetch
$ git pull
```
